// FAQ

The honest answers.

Is my data really private?

Yes. There are no accounts, no servers, and no telemetry. Your tasks, notes, and screen activity stay on your machine, encrypted at rest in the OS keychain. The AI models run locally — nothing is transmitted for us to even receive.

Does it need an internet connection?

No. Once the local models are installed, focus tracking, capture, coaching, and the copilot all work fully offline. Internet is only used if you opt into cloud integrations like Google Calendar or an optional API fallback.

Which AI models does it use?

Local LLMs through Ollama — Llama, Qwen, Phi, Gemma, or Mistral — and Whisper for voice. You can switch models, see RAM recommendations and accuracy stats, and install or delete them from inside the app. No terminal required.

Is there a Mac or Linux version?

Windows is the primary build — get it as a direct download or from the Microsoft Store. There's now an experimental Linux build too — a portable AppImage on the Linux download page; everything works except desktop-app distraction detection, which is limited on Wayland (the browser extension still covers the web). macOS is on the roadmap, with an Android companion and web dashboard further out.

What are the system requirements?

Windows 10/11, 8GB RAM for small models, 16GB recommended for the larger ones. A discrete GPU helps inference but isn't required — LockinRa picks a model that fits your hardware and tells you the trade-offs up front.

Download AppImage
v1.1.0 · Linux x86_64 · ~196 MB · single portable file, no installer

One self-contained file — nothing is installed system-wide, and you can delete it to uninstall. Your data lives in ~/.config/LockinRa.

Run it

After downloading, make it executable and launch it:

chmod +x LockinRa-1.1.0-linux-x86_64.AppImage
./LockinRa-1.1.0-linux-x86_64.AppImage

Or right-click the file → Properties → Permissions → Allow executing as program, then double-click it.

Hit a FUSE error? Some distros don't ship FUSE 2 by default. Either install it — sudo apt install libfuse2 (Debian/Ubuntu) or sudo dnf install fuse (Fedora) — or run it without FUSE:

./LockinRa-1.1.0-linux-x86_64.AppImage --appimage-extract-and-run

What to expect on Linux

This is an early Linux build. Almost everything works; one feature is limited depending on your display server.

Works
  • Tasks, voice & text capture
  • Focus sessions, Pomodoro, soundscapes
  • Notes, habits, analytics, calendar
  • Local AI (Ollama) & on-device voice (Whisper)
  • Browser-extension web distraction detection
  • Coach & Copilot, integrations
Limited
  • Desktop-app distraction detection works on X11 but is unreliable on Wayland (the default on many distros) — the web/extension path still covers browser distractions.
  • The OS media-session mini-player and the installed-apps Smart Launcher list are Windows-only.
  • No built-in auto-update yet — re-download to upgrade.

System requirements

  • A 64-bit Linux desktop (glibc-based, e.g. Ubuntu 22.04+, Fedora, Debian 12+).
  • 8 GB RAM for small local models, 16 GB recommended for larger ones.
  • A discrete GPU helps AI inference but isn't required.
